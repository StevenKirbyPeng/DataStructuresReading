一、作業目的

HW2 Territory Conquest Mini Game 是一個以二維地圖為基礎的領土征服模擬作業。玩家先建立一張 n 乘 n 的地圖，再根據輸入的 merge 操作將相鄰格子合併成較大的 region。接著，玩家選擇一個起始格子作為初始領土，並依照方向指令讓兵向上、下、左、右移動。每一步移動可能造成新區域被征服、兵停在邊界，或兵因走回已征服區域而退休。最後，系統會將結果輸出成 SVG 圖檔。

本作業的核心學習目標在於將資料結構與模擬流程整合。區域合併部分適合使用 Union Find 管理不同格子是否屬於同一集合；移動模擬部分則需要透過 active 與 owned 狀態追蹤目前仍能移動的兵與已被征服的地區。透過此作業，學生可以練習輸入解析、資料結構設計、狀態更新、流程控制與圖形化輸出。

重點概念是先用 Union Find 建立靜態 region，再用 Simulation 模擬動態 conquest，最後用 SVG 呈現 final map。

(三) Conquest 移動規則

情況規則下一步狀態

六、演算法設計

(一) Union Find 偽碼

find(x): if parent[x] == x: return x parent[x] = find(parent[x]) return parent[x] union(a, b): ra = find(a) rb = find(b) if ra == rb: return parent[rb] = ra

(二) 移動模擬偽碼

owned = start region active = all cells in start region for each direction d: nextActive = empty for each cell in active: target = move(cell, d) if target is out of bounds: nextActive.add(cell) else if target region is already owned: retire this troop else: conquer whole target region nextActive.add(all cells in target region) active = nextActive if active is empty: stop simulation

(三) 狀態集合說明

集合名稱意義更新時機
owned已被征服的 cell 或 region進入新 region 時更新
active本回合可以移動的兵所在 cell每一步結束後更新
retired走回已征服區域而停止的兵移動到 owned region 時產生
stayed因越界而停留的兵移動超出地圖時產生

七、SVG 輸出設計

SVG 是文字形式的向量圖，可以用 C++ 的 ofstream 產生，也可以直接使用瀏覽器開啟。輸出流程包含背景、格子填色、外框、不同 region 間的邊界，以及每格原始編號。

(一) SVG 繪製流程

1 畫白色背景 2 依照 owned 狀態填入每個 cell 的顏色 3 畫出整張 board 的外框 4 檢查每格右方與上方鄰居 5 若 root 不同則畫 boundary line 6 在每格中央畫出原始編號
同一 region 內部不畫分隔線，不同 region 之間才畫 boundary line，這是 SVG 輸出最重要的判斷。

八、複雜度分析

階段主要操作時間複雜度說明
Initialization建立 n × n 地圖O n 平方需要初始化所有 cell
MergeUnion Find 合併O k α nα n 為反阿克曼函數，實務上接近常數
Simulation依 p 步方向移動最壞 O p n 平方每一步可能需檢查許多 active cells
SVG Output畫格子、邊界、文字O n 平方每個 cell 至少處理一次
整體時間複雜度可視為 O n 平方 加上 O k α n 加上 O p n 平方。若 active cells 數量較少，實際執行時間會更快。

九、結論

本作業透過 Territory Conquest Mini Game 將資料結構與演算法整合成一個完整系統。Union Find 負責處理 region 合併，使多個格子可以被視為同一區域；simulation 負責處理動態征服流程，使每一步移動都能依照規則更新 active、owned、retired 與 stayed 狀態；SVG 則負責將最後結果視覺化，使使用者能清楚觀察征服結果與區域邊界。

透過本作業，學生不僅能熟悉 Union Find 的實際用途，也能學會如何將抽象資料結構應用於遊戲規則與圖形輸出中。整體而言，此作業具有良好的學習價值，能有效訓練問題分析、演算法設計、程式實作與結果呈現能力。
