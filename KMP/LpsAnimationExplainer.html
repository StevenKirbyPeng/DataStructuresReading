👉 LPS[-] = -
LPS[i] 的意思是：對於子字串 pattern[0..i]， 最長的「前綴」同時也是「後綴」的長度。
規則： 1. lps[0] = 0 2. 若 pattern[i] == pattern[len]：len++, lps[i] = len, i++ 3. 若 pattern[i] != pattern[len] 且 len != 0：len = lps[len - 1] 4. 若 pattern[i] != pattern[len] 且 len == 0：lps[i] = 0, i++

考試口訣：相同就加一，不同就往前跳，跳到 0 還不同就填 0。
